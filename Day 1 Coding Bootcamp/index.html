So Why Does This Page Exist?

The reason that I'm creating this page is to practice working with HTML. I used to really struggle with coding, and I thought that I had absolutely no future in it, but I want to give it another try because front-end developers and UI/UX designers do some really cool things through coding. Seeing amazing websites and portfolios inspires me to learn how to code. Maybe one day I will be able to code a really rad website for my art, design, or ceramics projects. It would also be fun to have a job which involves coding because the pay is good, you can work remotely, and it's flexible.
